Форма настройки обработки "Штрих-М: Весы с печатью этикеток"

Форма предназначена для настройки параметров работы весов с печатью этикеток, поддерживаемых драйвером «Штрих-М: Штрих-Принт». Обработка обслуживания позволяет настраивать следующие параметры устройства:

  • Логический номер. Для некоторых моделей весов необходимо также задать их логический номер.
  • Интерфейс. Тип интерфейса для связи.
  • Порт. Номер последовательного порта, к которому подключены весы.
  • Скорость. Скорость обмена данными с весами.
  • Таймаут. Таймаут для интерфейса RS232.
  • IP адрес. IP адрес весов.
  • UDP порт получателя. Входящий UDP порт весов.
  • Таймаут UDP. Таймаут для обмена по UDP протоколу.
  • Пароль. Пароль для доступа к системному меню и связи с КЭ по интерфейсу.
  • Быстрая загрузка. Предназначено для записи установки режима загрузки.

 

После изменения значений параметров следует нажать кнопку "ОК" для того, чтобы сохранить внесённые изменения, или кнопку "Отмена" для того, чтобы отказаться от их сохранения.
