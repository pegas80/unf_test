Основная форма монитора узла плана обмена

На этой форме сосредоточена вся информация, связанная с конкретным узлом этого плана обмена:

  • Состав плана обмена и информация о режиме авторегистрации объектов метаданных, входящих в состав.
  • Количество зарегистрированных изменений в узле.
  • Отдельные зарегистрированные объекты.

Узел плана обмена выбирается или при открытии формы в соответствующем параметре формы или в поле "Узел обмена".

В дереве зарегистрированных объектов при выборе объекта базы данных производится открытие следующих форм:

  • Для всех ссылочных типов данных открывается форма объекта.
  • Для всех регистров имеющих подчинение регистратору, открывается форма объекта - регистратора соответствующего набора записей записей регистра.
  • Для независимых регистров сведений открывается форма записи соответствующего регистра.

При проведении некоторых операций, которые могут оказаться длительными, по возможности предусмотрена выдача состояния процесса обработки и предусмотрена возможность прерывания операции пользователем.

При выборе узла плана обмена соответствующему этой базе данных, котекстное меню блокируется. Таким образом, для узла этой информационной базы допустимы только просмотр состава плана обмена и редактирование реквизитов узла.

 

Режим выделения строк дерева объектов множественный, таким образом для той или иной операции может быть выбран произвольный набор строк дерева. Все управление регистрацией и отменой регистрации объектов в узле сосредоточено в контекстном меню дерева объектов и применяется ко всем выделенным строкам. Команда  "Показать зарегистрированные объекты" заполняет дерево зарегистрированными объектами базы данных и запоминается в элементе дерева. Таким образом при обновлении или выполнения какой либо другой операцией с набором строк дерева происходит перечитывание конкретных зарегистрированных объектов (в случае большого количества зарегистрированных объектов это может привести к задержкам). Для отключения режима следует применить команду  "Скрыть зарегистрированные объекты". Команда  "Обновить данные обмена" в отличии от соответствующей команды в панели формы обновляет информацию только по выделенным строкам дерева объектов. Команда  "Зарегистрировать объекты" предназначена для регистрации объектов базы данных в узле плана обмена. В зависимости от набора выделенных строк производится регистрация либо всех объектов базы данных, либо по отдельным объектам метаданных. Команда   "Удалить регистрацию объектов" отменяет регистрацию объектов по такому же принципу, дополнительно возможна отмена регистрации отдельных объектов базы данных (если выделенныестроки находятся на уровне объектов баз данных). Команды  "Произвольная регистрация" и  "Произвольное удаление регистрации" производят  регистрацию и отмену регистрации по произвольным условиям. Подробнее смотри "Форма произвольной регистрации".

Командная панель формы

Команда  "Редактировать" позволяет перейти к форме редактирования реквизитов узла плана обмена. Команда  "Обновить" позволяет обновить все переменные данные дерева зарегистрированных объектов узла (количество зарегистрированных объектов и объекты баз данных, в том случае, если отображение этих объектов было предварительно включено командой "Показать зарегистрированные объекты". Команда  "Закрыть" закрывает форму.
