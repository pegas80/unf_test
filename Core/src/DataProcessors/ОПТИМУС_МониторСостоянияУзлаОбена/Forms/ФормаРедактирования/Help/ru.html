Форма редактирования реквизитов узла плана обмена

С помощью этой формы можно просмотреть и отредактировать реквизиты узла плана обмена.

Редактирование реквизитов узла плана обмена разрешено только при наличии права "Редактирование" у пользователя базы данных

Для узла плана обмена, соответствующей этой информационной базе данных реквизиты "Номер отправленного" и "Номер принятого" не выводятся на форму и недоступны для просмотра и редактирования

На форму по умолчанию выводятся следующие реквизиты узла плана обмена: 

  • Код
  • Наименование 
  • Номер отправленного сообщения
  • Номер принятого сообщения

Все остальные реквизиты могут быть размещены на форме с помощью команды  "Состав реквизитов".
